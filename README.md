# imp_tsp
Measure impulse response with time-stretched pulse (TSP) signal

## Usage
usage: python imp_tsp.py [-h] -l LENGTH -f SAMPFREQ [-o OUTPUT] [-i INPUT] [-p PLOT]
                         [-c CHANNEL] [-s SYNC] [-w WAVE] [-a AUDIO] [-d DIREC]
                         [-e EVAL]
